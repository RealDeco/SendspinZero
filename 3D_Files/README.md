## 3D Printable Files

3D printing files for the three versions of **Sendspin Zero**.

| File | Version | Description |
|-----|-----|-----|
| `SendspinZero-Toslink.3mf` | Optical Toslink | Enclosure for the optical Toslink version |
| `SendspinZero-Analog.3mf` | Analog Stereo | Enclosure for the analog stereo (3.5 mm minijack) version |
| `SendspinZero-Coax.3mf` | Coax SPDIF | Enclosure for the coaxial SPDIF version |

Each one has front for both with and without display or a button.

| File | Version | Description |
|-----|-----|-----|
| `SendspinZero-Control.3mf` | Analog version only | Enclosure for version with Rotary knob + a button |
| `SendspinZero-Display_Button.3mf` | Analog version only | Enclosure for slimmer desk version with long display and a button |
| `SendspionSupermini-Headphones.3mf` | Analog version only | Enclosure for Battery driven Headphone player |

**Notes**

- Printed in PETG and not tested in PLA, size "could" vary. (feedback welcome)
- The `.3mf` files were created with an **older version of Bambu Studio** to ensure broad compatibility.
- **STL files available**
